# 本月签证预约空位板块 — 需求与设计

> 面向：产品/评审，以及要实现的人。技术落地见 [[本月空位板块-实现规格]]；抓取原理见 [[爬取签证预约空位]]。

## 目标（一句话）

在 `/japan` 页「基础资料填写表单」**下方**加一个板块，用**月历**显示日本驻马来西亚大使馆
**代理代递交签证**（`event=12` / `plan=22`）本月哪天有空位、哪天没有。

## 需求清单（已与业主确认）

- 板块**一直显示**在表单下方，页面打开即见本月空位（无需先填表）。
- **仅日本页**显示（韩国页等不显示）。
- 固定显示 **代理代递交**这条业务线（`event=12` / `plan=22`，每次预约 ≤5 人）。
- 展示形式：**彩色月历网格** + 图例 + 一行「结论」重点提示。
- **‹ ›** 箭头切换上一月 / 下一月。
- 显示**最后更新时间**。
- **「刷新」按钮**：手动强制抓最新。
- **定时自动重新抓取**（建议每 10 分钟）。

## 展示设计规格

### 三态 + 两个辅助态

思路：**让「有空」跳出、「已满」沉下**——客户只关心哪天能约。

| 状态 | 符号 | 文案 | 浅色底 / 字色 | 处理 |
|---|---|---|---|---|
| 有空 | ○ | 可约 | `#e6f4ea` / `#1e7d34` | **最突出** |
| 仅剩 | △ | 仅剩 | `#fef3c6` / `#b45309` | 加紧迫感，可加粗 |
| 已满 | × | 已满 | `#f1f2f4` / `#9096a0` | **弱化，勿用警示红** |
| 过去 / 未放号 | · | — | 透明 / 极淡 | 不可点，几乎无存在感 |

### 三条硬性原则

1. **颜色不能单独用**（无障碍 + 黑白打印，本站有"打印清单"功能）：每格必须
   **色块 + 符号(○△×) + 文字** 三重编码。
2. **「已满」用中性浅灰，不要红**：满的天常常很多，整片红会焦虑、还盖过绿色；
   把饱和色留给「有空」「仅剩」，视线自然落到能约的日子。
3. **加一行「结论」**，别让客户自己数日历：
   > 🟢 最近可预约：**7/15（周三）** · 本月共 **12 天** 可约

   无空位时 → 「本月暂无开放的预约（大使馆尚未放号）」；
   有仅剩时可另提示 → `⚠️ 7/16 仅剩少量名额`。

### 可选加分项（非必须）

- 点「有空」的日子 → 新标签打开大使馆预约页并带参数
  （`event_id=12&event_plan_id=22&date=YYYY/MM/DD`），客户当场可去约。
- 「仅剩」格子加脉冲小圆点 / 加粗，强化"手慢无"。

## 重要业务语义（影响文案/说明）

- 空位按 `(event, plan)` **独立**：代理(22)与个人短期(9)同一天可能不同，本板块只代表代理线。
- 一次代理预约占 **1 个名额**、最多 **5 人**；>5 人需拆多次预约。
- 大使馆通常只放**当月及近期**；远期月份会显示"暂无开放"，属正常。
- 签证预约截止 = 预约日**前 3 个工作日下午 3 点**（可在板块下方加一行小字提醒）。
